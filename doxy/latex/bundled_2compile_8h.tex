Converts a string literal {\ttfamily s} into a format string that will be parsed at compile time and converted into efficient formatting code. Requires C++17 {\ttfamily constexpr if} compiler support.

{\bfseries{Example}}\+: \begin{DoxyVerb}// Converts 42 into std::string using the most efficient method and no
// runtime format string processing.

\end{DoxyVerb}
